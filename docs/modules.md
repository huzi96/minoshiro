#前端模块
-------------
###UI部分
```
controller:	
	载入文件及提示进度
	接口：
		监听事件on,progress和complete，事件event = {file: ""//载入文件名}
		载入模板文件、字体文件，以对象的方式传递给model
		
model:
	存储当前载入模板，绘制海报（不等于显示海报）
		
view:
	显示海报，监听鼠标
	
control_bar： 
	主控制窗口，在页面右侧。
	
	接口：
		与controller通信，处理初始载入、文字输入改变等事件
		提供不同的界面，实现不同的功能（文字输入、背景选择等)
	
```
